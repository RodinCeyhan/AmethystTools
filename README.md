# AmethystTools

AmethystTools is a powerful and customizable Paper plugin that adds special amethyst-based tools with advanced mining abilities and interactive GUI features.

Inspired by custom tool systems like those used in popular SMP servers, this plugin enhances survival gameplay with smarter and faster block breaking mechanics.

**Commands:**
- `/amethysttools <material> <type>`
- **Materials:** Wooden, Stone, Iron, Golden, Diamond, Netherite
- **Types:** Pickaxe, Shovel, Axe

**Menu:**
- The command `/amethysttools` opens a menu allowing you to access the tools.
